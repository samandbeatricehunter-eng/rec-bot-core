# REC Bot Menu Layout

## Top-Level `/menu`

- Rosters
- Manage My Team
- Standings & Stats
- REC Bank
- Media Center
- Help / Rules
- Admin Panel

Admin Panel appears only for authorized admins.

## Rosters

- View Rosters
- View Free Agents

## Manage My Team

- Schedule
  - Full season schedule
  - H2H opponents display as server @nickname
  - Strength of Schedule
    - Full season SOS
    - Remaining schedule SOS league-wide
  - Top 5 Rivalries
- Player Management
  - View user roster
  - Hire/manage trainers
  - Hire/manage scouts
  - Rookie-only draft class import workaround
  - One-time player development packages
  - Age resets
  - Dev upgrades when enabled
  - One-year contract extensions
  - 50% salary reductions
  - 50% bonus reductions
  - Legends/custom players when enabled
- Cap Management
- Talent Acquisition
  - Upcoming draft class scouting
  - Current free agency
  - Upcoming free agency
  - Trade menu
  - Custom player search
- Press Conference

## Standings & Stats

- League Standings
- User Records
- Playoff Race

## REC Bank

- Wallet & Savings
- Pending Purchases
- Purchase History

## Media Center

- Submit Highlight
  - Highlight posts trigger a DM asking whether to nominate for POTY
  - POTY categories: Passing TD, Rushing TD, Defensive Play, Defensive TD, Special Teams, Clutch Moment
- Vote Game of the Week
- Vote Game of the Year
  - Opens when advancing from regular season to playoffs
  - Closes when advancing to Super Bowl week
- Vote Play of the Year
  - Same window as GOTY
- View Previous Winners

## Help / Rules

- League Rules
- Economy Rules
- Scheduling Rules
- Import Rules
- Contact Admins

## Admin Panel

- Server Setup
- League Setup
- User / Team Linking
- Import Data
- Manual Entry
  - Draft picks entry belongs here
- Economy Admin
- Media Admin
- Audit Logs
