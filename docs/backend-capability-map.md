# Backend Capability Map

## Core Systems

- Identity: REC users, Discord accounts, EA accounts, future app accounts
- Server Setup: register servers, route channels, admin roles, setup validation
- League Management: leagues, seasons, phases, weeks, trust modes
- Team Ownership: user/team links, transfer ownership, history, EA reconciliation
- Imports: one week, season-to-week, companion export, rookie-only draft class import
- Scheduling: games, availability, confirmed time, ready state, result lock
- Economy: global wallet/savings, ledger, purchases, adjustments
- Media: highlights, DM nomination, GOTW/GOTY/POTY voting, winners
- Audit: setup, imports, manual changes, economy corrections
